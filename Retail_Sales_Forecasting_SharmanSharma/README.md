# Retail Sales Forecasting & Demand Analysis

## Project Overview

This project was developed as part of the XYLOFY AI Internship (Week 3 & Week 4).

The objective of this project is to analyze historical retail sales data, forecast future demand using multiple forecasting techniques, detect anomalies, perform customer/product segmentation, and visualize insights using an interactive Streamlit dashboard.

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn
- Statsmodels (SARIMA)
- Prophet
- XGBoost
- Streamlit

---

## Project Features

### Data Preprocessing

- Data Cleaning
- Missing Value Handling
- Date Conversion
- Feature Engineering

### Exploratory Data Analysis

- Monthly Sales Trend
- Region-wise Analysis
- Category-wise Analysis

### Forecasting Models

- SARIMA
- Facebook Prophet
- XGBoost

### Forecast Evaluation

- MAE
- RMSE
- MAPE

### Anomaly Detection

- Isolation Forest
- Z-Score Method

### Product Segmentation

- K-Means Clustering
- PCA Visualization

### Interactive Dashboard

- Business KPIs
- Category Analysis
- Region Analysis
- Monthly Sales Trend
